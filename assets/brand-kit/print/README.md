# Print Assets

## Source Files
All baseline logo variations:
- `polaris-logo-horizontal.svg` - Primary horizontal lockup
- `polaris-logo-horizontal-dark.svg` - For dark backgrounds
- `polaris-logo-vertical.svg` - Medium vertical lockup
- `polaris-logo.svg` - Compact vertical lockup
- `polaris-icon.svg` - Icon only

## Print Requirements

### High-Resolution Exports
For professional printing, export to:
- **PDF** - Vector format (preferred)
- **PNG** - 300 DPI minimum
- **EPS** - For design software compatibility

### Common Print Sizes

**Business Cards (3.5" × 2")**
- Logo width: ~2.5-3" at 300 DPI
- Export: ~900px wide at 300 DPI

**Letterhead (8.5" × 11")**
- Logo width: ~2-3" at 300 DPI
- Export: ~750px wide at 300 DPI

**Posters/Signage**
- Use SVG directly or export to PDF
- Minimum 300 DPI for any PNG exports

## Color Specifications
- **Primary**: #1a3a52 (RGB: 26, 58, 82) (CMYK: ~90, 68, 43, 45)
- **Accent**: #4a7c9e (RGB: 74, 124, 158) (CMYK: ~67, 39, 21, 4)
- **Secondary**: #5a6c7d (RGB: 90, 108, 125) (CMYK: ~60, 42, 33, 15)

## Usage Guidelines
- Maintain minimum clear space around logo (equal to height of icon)
- Never distort or skew logo
- Use on white or very light backgrounds (or dark variant on dark backgrounds)
- Minimum print size: 0.5" height for horizontal, 0.75" height for vertical
