# Polaris Advisor Logo Files

## Baseline Logo Suite

### Primary Logos

**`polaris-logo-horizontal.svg`** - Primary horizontal lockup
- Icon + text side-by-side
- Use for: website headers, email signatures, business cards, letterheads, banners
- Currently used on website

**`polaris-logo-vertical.svg`** - Medium vertical lockup
- Icon stacked above centered text (medium size)
- Font sizes: Polaris 90, ADVISOR 32
- Use for: hero images, posters, vertical formats

**`polaris-logo.svg`** - Compact vertical lockup
- Icon stacked above centered text (compact size)
- Font sizes: Polaris 22, ADVISOR 14
- Use for: square social posts, tight spaces, profile images

### Supporting Assets

**`polaris-icon.svg`** - Icon only (no text)
- Just the caliper/star symbol
- Use for: favicon, app icons, social avatars, small spaces

## Color Palette

- Primary: `#1a3a52` (Deep blue)
- Primary Light: `#2d5270`
- Accent: `#4a7c9e`
- Secondary Text: `#5a6c7d`
- White: `#ffffff`

## Typography

- Font: -apple-system, 'Helvetica Neue', Arial, sans-serif
- "Polaris": Weight 600 (Semibold)
- "ADVISOR": Weight 400 (Regular), uppercase, increased letter-spacing
