# Polaris Advisor Logo Files

## Baseline Logo Suite (Shared Sources)

### Primary Logos

**`polaris-logo-horizontal.svg`** - Primary horizontal lockup
- Icon + text side-by-side
- Use for: website headers, email signatures, business cards, letterheads, banners
- Currently used on website

**`polaris-logo-horizontal-dark.svg`** - Horizontal lockup (dark)
- Dark-background variant of the primary horizontal lockup

**`polaris-logo-vertical.svg`** - Medium vertical lockup
- Icon stacked above centered text (medium size)
- Font sizes: Polaris 90, ADVISOR 32
- Use for: hero images, posters, vertical formats

**`polaris-logo.svg`** - Compact vertical lockup
- Icon stacked above centered text (compact size)
- Font sizes: Polaris 22, ADVISOR 14
- Use for: square social posts, tight spaces, profile images

### Supporting Assets

**`polaris-icon.svg`** - Icon only (no text)
- Just the caliper/star symbol
- Use for: favicon, app icons, social avatars, small spaces

**`polaris-icon-dark.svg`** - Icon only (dark)
- Dark-background variant

**`polaris-icon-favicon.svg`** - Favicon-optimized icon (light)
- Simplified for 32×32 and 48×48

**`polaris-icon-favicon-light.svg`** - Favicon-optimized icon (light alt)
- Alternate light variant (use only if specified)

**`polaris-icon-tiny.svg`** - Tiny favicon icon (light)
- Optimized for 16×16

**`polaris-icon-tiny-light.svg`** - Tiny favicon icon (light alt)
- Alternate light variant (use only if specified)

## Usage Map (Shared Across Folders)

These SVGs are shared sources. Do not duplicate them in local asset folders.\n\n- If an SVG is used in more than one folder, keep it here and reference it from the local README.\n- Only keep SVGs inside `/web`, `/social-media`, `/email-signature`, `/print`, etc. when they are **unique to that folder’s PNG exports**.

## Color Palette

- Primary: `#1a3a52` (Deep blue)
- Primary Light: `#2d5270`
- Accent: `#4a7c9e`
- Secondary Text: `#5a6c7d`
- White: `#ffffff`

## Typography

- Font: Liberation Sans (use files in `../fonts/`)
- "Polaris": Weight 600 (Semibold)
- "ADVISOR": Weight 400 (Regular), uppercase, increased letter-spacing
